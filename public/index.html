Welcome to Messaging System

Available Endpoints:

  • /

    [GET] Index page for Messaging System.

  • /register

    [GET] Form to register a new account.
    [POST] Register function with password hashing and validation.

  • /verify

    [GET] Request using a valid token to verify account.

  • /login

    [GET] Form to login using credentials.
    [POST] Login function with validation.

  • /chat

    [GET] Messaging page for authenticated users.
